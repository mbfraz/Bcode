HTML y CSS

Nuestro favorito... y puede que el tuyo!

Juego preguntas y respuestas acerca de html, css y Java Scritp HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio.

Space Explorer

Una carrera infinita por el universo

Elite: Dangerous es un videojuego de simulación espacial y la cuarta entrega de la saga Elite. A bordo de una nave espacial el jugador explora una galaxia a escala 1:1 basada en la verdadera Vía Láctea. Es el primero de la serie Elite en ofrecer una experiencia multijugador masiva (MMO), en la que las acciones de los jugadores afectan directamente al universo persistente.
